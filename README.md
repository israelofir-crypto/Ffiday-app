# 🌅 שישי בבוקר

אפליקציית ווב קלילה שעוזרת להחליט מה עושים ביום שישי בבוקר — בכל עיר.

🌐 **גרסה חיה:** https://israelofir-crypto.github.io/Ffiday-app/

## איך זה עובד?

בוחרים עיר — פתח תקווה, תל אביב, ירושלים או חיפה (עם מאגר אטרקציות
אמיתיות לכל אחת), או מקלידים כל עיר אחרת ומקבלים הצעות כלליות עם
קישורי חיפוש במפות. אחר כך עונים על 5 שאלות קצרות:

1. **עם מי יוצאים?** — משפחה, זוגי, חברים או לבד
2. **איזו אווירה?** — טבע, אוכל וקפה, פעילות ספורטיבית או תרבות
3. **בפנים או בחוץ?**
4. **מה התקציב?**
5. **באיזה רדיוס?** — עד 3 ק"מ, עד 10 ק"מ או בלי הגבלה

לפי התשובות האפליקציה מדרגת את המקומות, מסננת לפי מרחק מהמרכז,
ומציגה את 3 ההתאמות הטובות ביותר — כולל טיפ לשישי בבוקר, תגית מרחק
וקישור למפות.

יש גם כפתור 🎲 **"הפתיעו אותי"** למי שמעדיף לתת לגורל להחליט.

## הרצה

אין תלויות ואין build — פשוט פותחים את `index.html` בדפדפן:

```bash
open index.html        # macOS
xdg-open index.html    # Linux
```

או מרימים שרת סטטי:

```bash
python3 -m http.server 8000
# ואז נכנסים ל- http://localhost:8000
```

## הוספת מקומות וערים

המקומות מוגדרים באובייקט `PLACES_BY_CITY` בתוך `index.html` (וההצעות
הכלליות לעיר מוקלדת ב-`GENERIC_PLACES`). כדי להוסיף מקום מוסיפים אובייקט
עם שם, תיאור, טיפ, קהל יעד (`who`), אווירה (`vibe`), מיקום (`setting`),
תקציב (`budget`), מרחק בק"מ מהמרכז (`km`) ושאילתת חיפוש למפות (`query`).
עיר חדשה מוסיפים למערך `CITIES` ולצידה רשימת מקומות ב-`PLACES_BY_CITY`.
