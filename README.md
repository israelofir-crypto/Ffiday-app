# 🌅 שישי בבוקר בפתח תקווה

אפליקציית ווב קלילה שעוזרת להחליט מה עושים ביום שישי בבוקר בפתח תקווה.

## איך זה עובד?

האפליקציה שואלת 4 שאלות קצרות:

1. **עם מי יוצאים?** — משפחה, זוגי, חברים או לבד
2. **איזו אווירה?** — טבע, אוכל וקפה, פעילות ספורטיבית או תרבות
3. **בפנים או בחוץ?**
4. **מה התקציב?**

לפי התשובות היא מדרגת רשימה של מקומות בילוי אמיתיים בפתח תקווה והסביבה
(הפארק הגדול, שוק העיר, מתחם סגולה, מוזיאון פ"ת לאמנות, מקורות הירקון ועוד)
ומציגה את 3 ההתאמות הטובות ביותר — כולל טיפ לשישי בבוקר וקישור למפות.

יש גם כפתור 🎲 **"הפתיעו אותי"** למי שמעדיף לתת לגורל להחליט.

## הרצה

אין תלויות ואין build — פשוט פותחים את `index.html` בדפדפן:

```bash
open index.html        # macOS
xdg-open index.html    # Linux
```

או מרימים שרת סטטי:

```bash
python3 -m http.server 8000
# ואז נכנסים ל- http://localhost:8000
```

## הוספת מקומות

כל המקומות מוגדרים במערך `PLACES` בתוך `index.html`. כדי להוסיף מקום חדש
מוסיפים אובייקט עם שם, תיאור, טיפ, קהל יעד (`who`), אווירה (`vibe`),
מיקום (`setting`), תקציב (`budget`) ושאילתת חיפוש למפות (`query`).
