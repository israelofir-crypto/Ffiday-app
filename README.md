# 🌄 בין הזמנים

אפליקציית ווב בעברית ובכיוון RTL שעוזרת למשפחות לתכנן יום בילוי בבין
הזמנים. היא עובדת גם כאפליקציה מותקנת בטלפון וכוללת חיפוש מקומות חי
באמצעות OpenStreetMap.

## איך זה עובד?

בוחרים נקודת יציאה מתוך הערים הנפוצות או מקלידים כל יישוב בארץ. לאחר
מכן עונים על שאלון קצר:

1. **עם מי יוצאים?** — משפחה, זוג, חברים או לבד
2. **גיל הילדים** — פעוטות, ילדים, נוער או גילים מעורבים
3. **סוג הבילוי** — טבע, אוכל כשר, אטרקציות או מורשת ועניין
4. **משך הבילוי** — כמה שעות, חצי יום או יום מלא
5. **בפנים או בחוץ**, **תחבורה**, **תקציב** ו**רדיוס**
6. בטיולי טבע: **מים**, **רמת קושי** ו**מסלול מעגלי**

האפליקציה מדרגת לפחות 10 הצעות לפי ההתאמה למשפחה, מסננת לפי המרחק,
ומציגה טיפים, מאפייני מסלול וקישור ישיר למפות. ערים שאין להן מאגר
מובנה מקבלות הצעות כלליות וחיפוש מקומי אוטומטי.

יש גם כפתור 🎲 **"הפתיעו אותי"** וכפתור להבאת אפשרויות חיות מהאינטרנט.
לפני סגירה מוצגת תזכורת לבדוק שעות, הרשמה, מזג אוויר, מצב מסלול,
כשרות והסדרי רחצה.

## הרצה

אין תלויות ואין build — פשוט פותחים את `index.html` בדפדפן:

```bash
open index.html        # macOS
xdg-open index.html    # Linux
```

או מרימים שרת סטטי:

```bash
python3 -m http.server 8000
# ואז נכנסים ל- http://localhost:8000
```

## הוספת מקומות וערים

המקומות מוגדרים באובייקט `PLACES_BY_CITY` בתוך `index.html` (וההצעות
הכלליות לעיר מוקלדת ב-`GENERIC_PLACES`). כדי להוסיף מקום מוסיפים אובייקט
עם שם, תיאור, טיפ, קהל יעד (`who`), אווירה (`vibe`), מיקום (`setting`),
תקציב (`budget`), מרחק בק"מ מהמרכז (`km`) ושאילתת חיפוש למפות (`query`).
עיר חדשה מוסיפים למערך `CITIES` ולצידה רשימת מקומות ב-`PLACES_BY_CITY`.
